# Токени

Статус: **порожньо.**

Дизайн-токени як єдине джерело правди для значень: колір, типографіка, простір,
радіуси, тіні, тривалості анімацій.

Токени первинні щодо компонентів — [`../components/`](../components/) споживає їх,
а не визначає власні значення.

## Що має тут з'явитися

- `tokens.json` — сирі значення у форматі, придатному і для дизайну, і для коду.
- Семантичний шар поверх примітивів: не `blue-600`, а `color-action-primary`.
  Це те, що дозволить пізніше додати темну тему без переписування компонентів.

## Відкрите

Формат і спосіб доставки токенів у код залежать від стеку, який ще не обрано.
Тому поки що тримаємо їх у нейтральному JSON, без прив'язки до фреймворку.
