# Career Evidence Tool

Система, що збирає робочі досягнення з доказами в момент, коли вони стаються.
Одне джерело правди про свою роботу замість реконструкції по пам'яті напередодні рев'ю.

Фундамент — збір і структурування матеріалу. Виходи (самооцінка для перформанс-рев'ю,
буллети для CV, заготовка кейсу для портфоліо) — змінні насадки на цей фундамент.

- **Для кого:** фахівці з досвідом (дизайнери, розробники, PM) у командах з регулярними
  циклами перформанс-рев'ю.
- **Платформа:** веб, desktop-first, далі адаптив до мобільного.
- **Дані:** зберігаються локально, у користувача.
- **Мова:** UA + EN.

**Статус:** фаза брифу й дизайну — коду продукту ще немає, стек не обрано.
Повний бриф: [CLAUDE.md](CLAUDE.md).

## Локальний запуск

Білду немає — це статичні файли. Достатньо будь-якого локального сервера:

```bash
python3 -m http.server 8000
```

Далі відкрити http://localhost:8000
