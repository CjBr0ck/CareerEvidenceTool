# Career Evidence Tool

Система, що збирає робочі досягнення з доказами в момент, коли вони стаються.
Одне джерело правди про свою роботу замість реконструкції по пам'яті напередодні рев'ю.

Фундамент — збір і структурування матеріалу. Виходи (самооцінка для перформанс-рев'ю,
буллети для CV, заготовка кейсу для портфоліо) — змінні насадки на цей фундамент.

- **Для кого:** фахівці з досвідом (дизайнери, розробники, PM) у командах з регулярними
  циклами перформанс-рев'ю.
- **Платформа:** веб, desktop-first, далі адаптив до мобільного.
- **Дані:** зберігаються локально, у користувача.
- **Мова:** UA + EN.

**Статус:** фаза брифу й дизайну. Коду продукту ще немає, стек не обрано.
Повний бриф — [CLAUDE.md](CLAUDE.md).

## Що показало дослідження

Перевірено 59 продуктів, чотири пройдено зсередини. Головний висновок:

> **Зафіксувати — не проблема. Ламається наступний крок: коли настає рев'ю, зібрати
> купу розрізнених скрінів, нотаток і цитат у готову самооцінку, буллети для CV чи
> кейс портфоліо. Цього не зробив ніхто.**

Прямих конкурентів приблизно вісім, і в жодного немає користувачів. Найближчий
попередник — Bragdocs — за п’ять років роботи зібрав 1 400 людей, які зробили хоча б один
запис, і закрився. Зрив стається не на реєстрації, а **на другому записі**, бо
з накопиченого нічого придатного не збирається і повертатись нема по що.

Звідси головне для планування: **основні зусилля йдуть не в інбокс, а в збірку.**
Швидка форма є у всіх конкурентів і нікому не допомогла.

Деталі, таблиці й джерела — [research/research.md](research/research.md).

## Прототип-заглушка

У корені лежать `index.html`, `styles.css`, `script.js` — статична сторінка-заглушка
з першого комміту. Це **не продукт**, а плейсхолдер, поки триває фаза дизайну.

### Локальний запуск

Білду немає — це статичні файли. Достатньо будь-якого локального сервера:

```bash
python3 -m http.server 8000
```

Далі відкрити http://localhost:8000

### Деплой

Хоститься на GitHub Pages з гілки `main`, корінь репозиторію.
Кожен push у `main` публікується автоматично протягом ~1 хвилини.
