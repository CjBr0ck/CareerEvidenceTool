# Компоненти

Статус: **порожньо.**

Окремі елементи інтерфейсу: специфікація кожного — стани, варіанти, поведінка,
доступність. Значення беруться з [`../tokens/`](../tokens/), не задаються локально.

## Ймовірний склад для MVP

Похідні від екранів у [`../wireframes/`](../wireframes/):

- Поле швидкого вводу (інбокс) — головний компонент продукту.
- Картка запису у списку.
- Тег і селектор тегів.
- Блок доказу — чотири типи: посилання, цитата, файл/скріншот, числова метрика.
- Поле метрики «до / після / одиниця».
- Пошук і фільтри.
- Порожні стани — важливі: продукт починається з порожнього списку.

Кожен компонент документує стани, а не лише вигляд за замовчуванням:
hover, focus, disabled, помилка, завантаження, порожньо.
